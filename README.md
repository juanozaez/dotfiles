# Dotfiles #

Set of files to quickly set up your development applications on MacOs.

#### Zsh
Install Zsh
Create /etc/zshenv (to move zsh from root dir)
Install Zim

#### Vim
Path = ~/.vimrc

#### Tools for shell
Install Exa
Install FZF
Install jq

### Files
#### Zsh
.zshenv: environment variables
.zshrc: shell configuration 
.zlogin: start of login shell
